FB Photools [How to use]

Original size

Useful to find the image of a Facebook profile in its larger size, including locked images for viewing.

Example → https://fbcdn-profile-a.akamaihd.net/hprofile-ak-ash3/c66.66.828.828/s160x160/536989_10151266339362308_1763871769_n.jpg

Source photo

Useful to find the source (page or profile) of any image. Remember that to access a post you must have access permission, and so this feature works best with photos of Facebook pages - are public.

Example → https://fbcdn-sphotos-d-a.akamaihd.net/hphotos-ak-ash4/1239043_10151640433322308_88764962_n.jpg

FB Photools is not a part of Facebook. There is no link between this project and the company.
